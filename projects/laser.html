Wood Laser Cutting Machine

  • Tech Stack: CNC Controller, Gcode, Embedded C, Open Source

1. As i have hobby of RC planes and drones for the frame structure of plane i have developed the laser cutting machine which cuts the wood very accuratly and so need less balancing for the final structure of RC planes.

2. The input for laser cutting was the file which is generated from the software called inkscape. In that software we have to give input image and we have to set the dimensions of the image for cutting.

3. Then we have to load the image file in the software called as Mach3 CNC it will generate the GCODE file which is transmitted to the 3 axis controller board using RS232.
